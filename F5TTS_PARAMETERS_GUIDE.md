# F5-TTS: Оптимизация параметров для точной генерации голоса

## Краткое резюме: Лучшие параметры для максимальной точности

```python
presenter_speach, sr, spec = f5tts.infer(
    ref_file="anton.wav",
    ref_text="",
    gen_text=accentizer.process_all(text),
    file_wave="podubskiy_gen1.wav",
    cfg_strength=4.5,        # ⬆️ Увеличить для большей точности голоса (3-7)
    seed=42,
    nfe_step=64,             # ⬆️ Увеличить для лучшего качества (32-128)
    remove_silence=True,
    target_rms=0.1,          # ✓ Оптимально
    cross_fade_duration=0.15,# ✓ Оптимально
    speed=1.0,               # ✓ Оптимально для естественности
    sway_sampling_coef=-1,   # ✓ Рекомендуемое значение
    fix_duration=None,       # ✓ Оптимально (автоматическая длина)
)
```

---

## Подробное описание каждого параметра

### 1. **cfg_strength** (Classifier Free Guidance)
**Текущее значение:** `3`  
**Оптимальный диапазон:** `2.0 - 7.0`  
**Рекомендация для максимальной точности:** `4.5 - 5.5`

| Значение | Эффект |
|----------|--------|
| 0.5 - 1.5 | Менее близко к голосу; больше вариативности, нестабильность |
| 2.0 - 3.0 | **Текущее значение** - хороший баланс, но можно улучшить |
| 3.5 - 5.5 | 🎯 **Оптимально для точности** - точное копирование голоса, высокая стабильность |
| 6.0 - 7.0 | Очень строгое соответствие; может быть дергано, местами артефакты |
| 8.0+ | Прослеживаются глюки и артефакты |

**Совет:** Начните с `4.5`, если результаты слишком близко к оригиналу и звучат странно, уменьшите до `4.0`.

---

### 2. **nfe_step** (Number of Function Evaluations)
**Текущее значение:** `64`  
**Диапазон:** `16 - 128`  
**Оптимум для качества:** `64 - 128`

| Значение | Качество | Скорость | Рекомендация |
|----------|----------|----------|--------------|
| 16 | Низкое, артефакты | ⚡ Очень быстро | Только для тестирования |
| 32 | Среднее | 🚀 Быстро | Для быстрого результата |
| 64 | **Хорошее** ✓ | ⚙️ Нормально | 🎯 Оптимальный баланс |
| 96 | **Очень хорошее** | ⏱️ Медленнее | Для критичного качества |
| 128 | **Отличное** | 🐢 Медленно | Максимум качества (редко нужно) |

**Совет:** Текущее значение `64` уже хорошо. Увеличьте до `96` для критичных проектов.

---

### 3. **seed**
**Текущее значение:** `42`  
**Рекомендация:** Оставить или использовать фиксированное значение

- `42` - ваш текущий выбор, хорош для воспроизводимости
- Любое фиксированное число - повторяемые результаты
- `None` или случайный - разнообразие, но не воспроизводимо

**Совет:** Для итераций оставьте `42`. Когда результат устраивает, можно попробовать другие значения для вариативности.

---

### 4. **target_rms** (Root Mean Square - громкость)
**Текущее значение:** не указано (используется по умолчанию `0.1`)  
**Диапазон:** `0.05 - 0.3`

| Значение | Громкость |
|----------|-----------|
| 0.05 | Тихо |
| 0.1 | 🎯 **Оптимально** (среднее) |
| 0.15 | Выше среднего |
| 0.2 - 0.3 | Громко |

**Совет:** `0.1` - оптимально. Не менять без необходимости.

---

### 5. **cross_fade_duration**
**Текущее значение:** не указано (по умолчанию `0.15` сек)  
**Рекомендация:** `0.15` секунд

- Время перехода между исходным и сгенерированным аудио
- `0.1 - 0.2` - оптимальный диапазон
- Избегать значений < 0.05 (слышны скачки)

**Совет:** Оставить `0.15`, это хороший баланс.

---

### 6. **speed**
**Текущее значение:** не указано (по умолчанию `1.0`)  
**Диапазон:** `0.5 - 2.0`

| Значение | Эффект |
|----------|--------|
| 0.5 - 0.8 | Медленная речь |
| 1.0 | 🎯 **Нормальная** (рекомендуется) |
| 1.2 - 1.5 | Быстрая речь |
| 2.0+ | Очень быстрая |

**Совет:** Оставить `1.0` для естественности.

---

### 7. **sway_sampling_coef**
**Текущее значение:** не указано (по умолчанию `-1`)  
**Рекомендация:** `-1` или `0`

- `-1` - отключено (рекомендуется)
- `0 - 1` - добавляет вариативность; экспериментально

**Совет:** Оставить на `-1`.

---

### 8. **fix_duration**
**Текущее значение:** `None` (автоматическая длина)  
**Рекомендация:** Оставить `None`

- `None` - автоматически подгоняет длину
- Число (сек) - фиксирует продолжительность

**Совет:** `None` - оптимально, не менять.

---

### 9. **remove_silence**
**Текущее значение:** `True`  
**Рекомендация:** `True` (оставить)

Удаляет тишину в начале и конце. Улучшает качество финального файла.

---

## Тестирование и экспериментирование

### Стратегия подбора параметров:
1. **Шаг 1:** Используйте `cfg_strength=4.5, nfe_step=64` (текущий + небольшой апгрейд)
2. **Шаг 2:** Если результат хорош → увеличьте `nfe_step` до 96
3. **Шаг 3:** Если звук странный → уменьшьте `cfg_strength` до 4.0
4. **Шаг 4:** Если нужна максимальная точность → `cfg_strength=5.0, nfe_step=96-128`

### Совет по подготовке эталонного аудио:
- **Длина:** 10-12 секунд (не более 12s)
- **Тишина:** Оставьте ~1 сек тишины в конце
- **Качество:** Чистый звук без фона
- **Содержание:** Разборчивая речь
- **`ref_text`:** Пусто или точный текст из аудио (если знаете)

---

## Дополнительные советы

### Для русского языка (используется accentizer):
- Убедитесь, что `accentizer.process_all(text)` работает корректно
- Проверьте правильность расставления ударений в тексте
- Ударения влияют на звучание, даже более чем `cfg_strength`

### Проблемы и решения:

| Проблема | Решение |
|----------|---------|
| Звук не похож на оригинал | ⬆️ Увеличить `cfg_strength` до 5.0 |
| Странные артефакты | ⬇️ Снизить `cfg_strength` до 3.5 или 4.0 |
| Низкое качество | ⬆️ Увеличить `nfe_step` до 96 |
| Слишком медленно | ⬇️ Снизить `nfe_step` до 32 или 64 |
| Тихий звук | ⬆️ Увеличить `target_rms` до 0.15 |
| Громкий звук | ⬇️ Снизить `target_rms` до 0.05 |

## Финальная рекомендация

**Начните с этого кода:**
```python
presenter_speach, sr, spec = f5tts.infer(
    ref_file="anton.wav",
    ref_text="",
    gen_text=accentizer.process_all(text),
    file_wave="podubskiy_gen1.wav",
    cfg_strength=4.5,        # ← Основное изменение для точности
    seed=42,
    nfe_step=96,             # ← Улучшение качества
    remove_silence=True,
    target_rms=0.1,
    cross_fade_duration=0.15,
    speed=1.0,
)
```

Если результаты будут слишком похожи и звучат странно, снизьте `cfg_strength` до 4.0. Если нужна максимальная точность - поднимите до 5.0-5.5.
