# Курс по комбинаторике и вероятностному методу

## 1. Гиперграфы
- **Определение**: гипер­графом называется пара $H=(V,E)$, где  
  - $V$ – конечное множество вершин,  
  - $E \subseteq \mathcal P(V)$ – набор ребер (подмножества $V$).  
- **n‑однородный гиперграф**: все ребра имеют размерность $n$:  
  $$\forall e\in E:\ |e|=n.$$
- Пример: обычный граф – 2‑однородный гипер­граф.

## 2. Хроматическое число гипер­графа
- Для гипер­графа $H=(V,E)$ и целого числа $k$ определяем **$k$-раскраску** как функцию $\chi: V \to [k]$.  
- Раскраска называется **правильной**, если в каждом ребре встречаются хотя бы два разных цвета.  
- Хроматическое число $\chi(H)$: минимальное $k$, при котором существует правильная раскраска.

## 3. Свойство B и функция $m(n)$
- Гипер­граф имеет **свойство B**, если он 2‑раскрашиваем (т.е. $\chi(H)=2$).  
- Минимальное число ребер, при котором существует n‑однородный гиперграф без свойства B:
  $$m(n) = \min\{|E| : H=(V,E)\text{ – }n\text{-однородный},\, \chi(H)>2\}.$$
- Задача: найти точное значение $m(n)$. Известны лишь оценки.

## 4. Вероятностный метод
### 4.1. Случайная раскраска
- Каждый вершине независимо присваивается цвет «красный» или «синий» с вероятностью $\tfrac12$.  
- Для фиксированного ребра $e$ событие  
  $$A_e = \{\text{ребро } e \text{ одноцветное}\}.$$
- Вероятность:
  $$\Pr(A_e)=\frac{2}{2^{|e|}}=2^{1-|e|}=2^{1-n}.$$

### 4.2. Условие «нет одноцветных ребер»
- Событие $B = \bigcap_{e\in E}\overline{A_e}$ – все ребра разноцветные.  
- По неравенству объединения:
  $$\Pr(B)=1-\Pr\!\left(\bigcup_{e\in E} A_e\right)
          \ge 1-\sum_{e\in E}\Pr(A_e).$$
- Если $|E|<2^{\,n-1}$, то  
  $$\sum_{e\in E}\Pr(A_e)=|E|\cdot 2^{1-n}<1,$$
  следовательно $\Pr(B)>0$. Значит, существует правильная раскраска.

### 4.3. Нижняя оценка
- Из выше: если $|E|<2^{\,n-1}$, гипер­граф имеет свойство B.  
- Поэтому  
  $$m(n)\ge 2^{\,n-1}.$$

## 5. Верхняя оценка
### 5.1. Полный n‑однородный гиперграф на $V=\{1,\dots,2n-1\}$
- Обозначим  
  $$V = [2n-1],\qquad E = \binom{[2n-1]}{n}.$$
- Любой раскраска двух цветов содержит хотя бы одно одноцветное n‑подмножество:  
  - В раскраске красных вершин ровно $r$ элементов, где $0\le r\le 2n-1$.  
  - Если $r<n$, то в синем множестве есть $\ge n$ вершины → одноцветный синий набор.  
  - Аналогично при $r>n$. При $r=n$ сразу одноцветный красный набор.

### 5.2. Следствие
- Гипер­граф $H=(V,E)$ **не имеет свойства B**.  
- Поэтому  
  $$m(n)\le |E|=\binom{2n-1}{\,n}.$$

## 6. Итоговые оценки
$$2^{\,n-1}\;\le\; m(n)\;\le\;\binom{2n-1}{\,n}
  \quad\text{(для всех } n\ge 2).$$
- Нижняя граница достигается при малом числе ребер.  
- Верхняя граница получается из полного гиперграфа на $2n-1$ вершинах.

## 7. Дальнейшие направления
- Сокращение разрыва между нижней и верхней оценками с помощью более тонких вероятностных инструментов (неравенства концентрации, Chernoff‑bounds).  
- Исследование точного значения $m(n)$ – открытая задача в комбинаторике.  
- Связь с задачами экстремальной теории графов и гипер­графов.

## 8. Ключевые формулы
- $\Pr(A_e)=2^{1-n}$.  
- $\Pr(B)\ge 1-|E|\cdot 2^{1-n}$.  
- $m(n)\ge 2^{\,n-1}$.  
- $m(n)\le \binom{2n-1}{\,n}$.
